Nuxeo Platform 5.7.1

Nuxeo Platform 5.7.1 is now available for download (1). This is the first Fast Track version, that uses the newly launched versioning system (2), with FT on odd numbered releases and Long Term Support on even numbered releases.

Following is the list of new features and improvements included in this new release:

  • Nuxeo Platform
  • DAM
  • Marketplace Packages
  • Nuxeo Visible Content Store (VCS), the Repository Storage Mechanism
  • Upgrade Notes (3)
  • Known Issues

(1) Download Nuxeo 5.7.1 FastTrack: http://nuxeo.github.io/downloads.html
(2) New LTS and FastTrack release cycle announcement: http://www.nuxeo.com/blog/updates/2013/05/release-cycle/
(3) Nuxeo 5.7.1 release notes: [PDF download URL from doc.nuxeo.com]
